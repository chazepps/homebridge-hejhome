Hejhome

for Web

회원가입은 앱스토어나 플레이스토어에서 헤이홈 어플을 다운 받아 가능합니다. 헤이홈 for Web은 구글 크롬 브라우저 전용 서비스입니다.

로그인 정보는 Homebridge 저장 공간에 저장되며 비밀번호는 저장하지 않습니다. 자동 로그인은 오래 로그인된 상태를 유지하기 위해 항상 켜집니다.

Hejhome 설정

Hejhome Web 로그인 정보를 Homebridge 저장 공간에 보관하고, 발견한 장비를 Home 앱에 보여줍니다.
